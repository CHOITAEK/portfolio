WEB

생활코딩의 웹 애플리케이션 만들기를 완주하고 새로운 WEBn수업을 복습차원에서 다시 보고 시작함.

웹은 HTML,Javascript,CSS,PHP,데이터베이스(mysql 등등)으로 구성되어 있음

HTML

WEB1(HTML)에서는 대부분 HTML과 서버, 웹호스팅에 대해 다룸. 기존의 웹 애플리케이션 만들기 수업하고 차이점은 각 part별로 자세히 다르고 서버를 다른 JS, php, mysql보다 먼저 함으로써 웹의 구동원리를 잘 알려줌.

CSS

web2(CSS)에서는 밋밋한 정보인 HTML를 꾸며줌으로서 디자인을 해주는 CSS를 배움. 여기서 CSS의 중복을 최소화하기 위해 ~.css 파일을 만듬. linkTAG를 이용하여 활용을 극대화함.
